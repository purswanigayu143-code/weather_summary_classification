# 🌦 Weather Summary Classification App

A Machine Learning based Weather Summary Prediction App built using **Python, Scikit-learn and Streamlit**.

## 🚀 Project Overview

This application predicts weather summaries based on weather parameters like:

- Temperature
- Humidity
- Wind Speed
- Visibility
- Pressure
- Precipitation Type

The trained ML model classifies weather conditions such as:
- Overcast
- Cloudy
- Foggy
- Clear etc.

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Joblib
- Streamlit
